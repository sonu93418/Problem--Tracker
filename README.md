### Adding a Problem
1. Enter the problem name in the input field
2. (Optional) Add the problem URL
3. Click **Add Question** or press Enter
4. Problem appears in the **To-Do** section

### Managing Problems
- **Change Status** - Use the dropdown to mark as Pending, Done, or Revise
- **Delete Problem** - Click the trash icon
- **Open Problem** - Click the **Open Problem** button to visit the URL
- **Search** - Use the search bar to filter problems
- **Sort** - Toggle between Newest/Oldest
